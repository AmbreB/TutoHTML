Cours sur le HTML

Les balises

L’Hypertext Markup Language, généralement abrégé HTML, est le format de données conçu pour représenter les pages web. Il est composé de balises permettant de structurer le contenu des pages.

Balises ouvrantes et fermantes

<div> (balise ouvrante)
Contenu
</div> (balise fermante)

Balises autofermantes

<img/>(balise autofermante)

Attributs de balises

Les attributs permettent de préciser les propriétés des éléments HTML. Pour le cas de <img/>, l'attribut « src » permet de préciser la source (provenance) de l'image ou le « alt ». Si ce dernier attribut est obligatoire, il en existe des facultatifs comme l'attribut « title » permettant de renseigner sur la nature ou le contenu de l'image avec une infobulle. Ces différents attributs peuvent être additionnés et sont séparés par des espaces. Ils fonctionnent sur un système de clé/valeur. <img src="fichier.jpg" alt="Description de l'image" title="Mon image"/>
src="fichier.jpg" (Ici la clé est 'src' et la valeur est 'fichier.jpg')

Les selecteurs id et class

Les sélecteurs permettent d'interagir avec le contenu avec CSS et JS.

L'attribut « id » identifie un élément de manière unique, alors que « class » permet une mise en forme (avec CSS par exemple)

class="class1 class2 class3"
id="menu"

Autres balises

Il existe beaucoup d'autres balises, elles ont toute un sens sémantique pour le navigateur et permettent de mieux structurer notre page. <div> → Bloc
<span> → Bout de texte inline
<br> → Retour chariot
<nav> → Element de navigation
<blockquote> → Citation
<p> → Paragraphe

Cours sur le CSS

Syntaxe et règles

Les feuilles de style en cascade, généralement appelées CSS de l'anglais Cascading Style Sheets, forment un langage informatique qui décrit la présentation des documents HTML. On appelle cela ainsi, car le navigateur va lire le document de haut en bas. Ceux qui ont la plus grande priorité sont en bas. En voici la syntaxe : selecteur {
propriété:valeur;
}

Les selecteurs

Les classes, qu'on a abordées plus haut, vont permettre de styliser précisément un ou plusieurs éléments de la page. Pour cela, on va utiliser les sélecteurs. En CSS désigne une class avec un (.) et un id avec un (#). Dans mon HTML :
<p class="class1">Hello</p>

Dans mon CSS :
.class1 {
color:red;
}

Les pseudos sélecteurs

Les pseudos sélecteurs permettent de sélectionner un élément selon son état (cliqué, visité...etc). Ils s'écrivent juste après l'élément avec ':pseudo-selecteur' Exemple :

/* lien non-visité*/
a:link {
    color: #FF0000;
}

/* lien visité */
a:visited {
    color: #00FF00;
}

/* curseur sur le lien */
a:hover {
    color: #FF00FF;
}

/* lien selectionné */
a:active {
    color: #0000FF;
}

Il est possible d'utiliser cette propriété pour émuler le comportement des navigateurs qui n'appliquent pas correctement la spécification du modèle de boîte CSS.

Centrer un bloc en CSS

Pour centrer un bloc 'div' en css, il faut lui définir une taille 'width' et lui appliquer la propriété 'margin' avec pour valeur 'auto' element {
width:200px;
margin:auto;
}

Computer science

DRY : Don't Repeat Yourself !

Ne vous répétez pas (don’t repeat yourself en anglais, aussi désigné par l’acronyme DRY) est une philosophie en programmation informatique consistant à éviter la redondance de code au travers de l’ensemble d’une application afin de faciliter la maintenance, le test, le débogage et les évolutions de cette dernière.

SoC : Separation of Concerns

Approche pour concevoir une solution ou un service informatique, qui divise le problème en morceaux pouvant être résolus indépendamment. Cette approche sépare le « quoi » de la conception du « comment » de la conception.
